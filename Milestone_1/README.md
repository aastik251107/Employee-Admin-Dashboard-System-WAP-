# Random User Directory

A simple frontend project that fetches user data from the [Random User API](https://randomuser.me/) and displays it in a clean card layout.

## Features

- Fetch random users from API
- Search users by name, email, or country
- Filter users by country
- Sort users by name or country
- Responsive layout for desktop and mobile

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Array Higher-Order Functions:
  - `map()`
  - `filter()`
  - `sort()`
